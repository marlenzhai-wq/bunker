# Bunker bot: әкімшілік жүйесі

Бұл жоба бір бас админ мен SQLite-та сақталатын жүргізушілер жүйесін іске
асырады. Рөл тексерісі әр басқару командасы орындалғанда базаға қайта сұрау
жасайды, сондықтан `/removehost` орындалғаннан кейін ескі жүргізуші келесі
әрекетін бірден орындай алмайды.

## Баптау және іске қосу

1. `config.py` ішіндегі `SUPER_ADMIN_ID = 0` мәнін өз Telegram ID-ңызға
   ауыстырыңыз. Бұл — боттағы жалғыз бас админ.
2. PowerShell-де бот токенін беріңіз:

   ```powershell
   $env:BOT_TOKEN = "<bot-token>"
   ```

3. Тәуелділіктерді орнатып, ботты іске қосыңыз:

   ```powershell
   py -m pip install -r requirements.txt
   py main.py
   ```

`bunker.db` жанында автоматты түрде жасалады; осы файл жүргізушілер мен ойын
күйін бот қайта іске қосылғанда да сақтайды.

## Командалар

Бас админ:

- `/addhost <Telegram ID>` немесе пайдаланушы хабарламасына Reply
- `/removehost <Telegram ID>` немесе пайдаланушы хабарламасына Reply
- `/hosts`

Бас админ және белсенді жүргізуші:

- `/startgame`, `/go` (сондай-ақ `го`), `/vote`, `/next`, `/endgame`

Әкімшілік және ойын-басқару оқиғалары `bunker.audit` логгеріне жазылады.
